    Basic Example¶

 KeycloakOperation.GET_USERS,
 "{\"realm\": \"quarkus\"}"
)

    Testing Instructions¶

    Compile the Project¶

    cd /Users/sshaaf/git/java/keycloak-mcp-server
mvn clean compile

    Build the JAR¶

    mvn package

    Run the Server¶

    java -jar target/keycloak-mcp-server-0.2.0-runner.jar

    Test with MCP Client¶

    Configure your MCP client (Goose, Claude Desktop, etc.) and try: 1. Get all users: operation: GET_USERS, params: {realm: "quarkus"} 2. Create a user: operation: CREATE_USER, params: {...} 3. Add role to user: operation: ADD_ROLE_TO_USER, params: {...}
